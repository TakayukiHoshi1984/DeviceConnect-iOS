AllJoynデバイスプラグインをビルドするためには以下のステップが必要になります。

1) AllJoyn iOS SDK v15.04b の zip をダウンロード：

  https://allseenalliance.org/releases/alljoyn/15.04/alljoyn-15.04.00b-osx_ios-sdk.zip

2) zipを伸展し、SDK 以下の darwin/ というディレクトリを AllJoyn/ に移動する：

  <SDK_ROOT>/build/darwin/  →  <ALL_JOYN_ROOT>/deps/AllJoyn/darwin/
